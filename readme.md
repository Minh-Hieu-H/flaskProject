# Project Python
## Hướng dẫn chạy
- Cài môi trường python trực tiếp trên từng project nhỏ sử dụng
   `python -m venv .venv`
- Thực hiện chạy server trực tiếp trên local sử dụng 
  `flask run`
- Cài đặt các thư viện liên quan trong file **requirement.txt**
- Sử dụng Docker Desktop trên window để tạo Image và build các project trên Docker sử dụng câu lệnh 
  `Docker build -t rest-api-flask-python . `
## Nội dung tìm hiểu về python
 - Python cơ bản

 - Sử dụng  Swagger UI  tại `http://localhost:5000/swagger-ui`
